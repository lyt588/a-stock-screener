# A股股票筛选工具

基于 AKShare 的 A 股选股工具，提供命令行版和 Streamlit 网页版。

## 筛选条件（可在网页版自由调整）

| 条件 | 默认值 | 说明 |
|------|--------|------|
| 涨跌幅 | > 3% | 当日上涨超过设定值 |
| 成交额 | > 5亿元 | 流动性过滤 |
| 价格区间 | 5~80 元 | 排除仙股与高价股 |
| MA20 向上 | 开启 | 今日20日均线 > 5日前20日均线 |
| 最多显示 | 50 只 | 结果按涨跌幅降序截取 |

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2a. 网页版（推荐）

```bash
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`。在左侧面板调整参数，点击「开始筛选」即可。结果支持一键导出 CSV。

### 2b. 命令行版

```bash
python main.py
```

## 运行流程

1. 拉取沪深 A 股全量实时行情（约 5000 只）
2. 用涨幅 / 成交额 / 价格做快速初筛（本地计算，秒级完成）
3. 对初筛结果逐一下载近 60 日历史 K 线，验证 MA20 趋势（可选，初筛越多越慢）
4. 展示结果表格 / 导出 CSV

## 注意事项

- 需要能访问 AKShare 数据源（境内网络即可）
- 建议在 **交易日收盘后（15:30 以后）** 运行，数据最为完整
- 网页版实时行情有 5 分钟缓存，均线数据有 1 小时缓存，刷新页面不会重复请求
- Python 版本要求：>= 3.10
